# Netflix Clone 
React-based SPA inspired by Netflix . Built using React , Vite , Firebase Authentication , and TMDB API with responsive UI and movie trailer integration.

## Features
- User Authentication using Firebase
- Movie and TV data from TMDB API
- Responsive UI
- Movie Title cards and banners
- React Router navigation
- Dynamic movie fetching

  ## Tech Stack
  - React
  - Vite
  - Firebase
  - TMDB API
  - HTML , CSS , JS
